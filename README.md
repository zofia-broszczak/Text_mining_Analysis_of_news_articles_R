# Text Mining Analysis of News Articles

This project applies unsupervised text mining methods to the AG’s News Topic Classification Dataset to explore the structure of news content through clustering, topic modeling, and network analysis.

## Project Overview

The analysis focuses on three perspectives:
- Clustering of article titles to identify document-level similarity  
- Topic modeling of descriptions to uncover latent themes  
- Word co-occurrence network analysis to study linguistic relationships  

The predefined news categories (World, Sports, Business, Sci/Tech) are used only for interpretation, not for model training.
